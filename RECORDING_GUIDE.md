# Portfolio Recording Guide for LinkedIn

## OpenScreen Settings

### Recording Setup
1. **Resolution:** 1920x1080 (1080p)
2. **Aspect Ratio:** 16:9 (landscape)
3. **Frame Rate:** 60fps for smooth animations
4. **Background:** Gradient or solid dark color (makes your site pop)
5. **Motion Blur:** Enable (makes panning smoother)

### Recording Tips

#### Before Recording
- [ ] Close unnecessary browser tabs/windows
- [ ] Hide bookmarks bar (Cmd+Shift+B in Chrome)
- [ ] Open portfolio in **Incognito/Private mode** (clean, no extensions)
- [ ] Set browser zoom to 100%
- [ ] Clear browser console (F12, then close it)
- [ ] Prepare your scroll path (know what you'll show)

#### During Recording
- [ ] Move mouse **slowly and deliberately**
- [ ] Pause 2-3 seconds on each section
- [ ] Show interactions (hovers, clicks) smoothly
- [ ] Don't rush - slower is better for viewers

#### What to Showcase (30-60s)
1. **Hero (0-7s)**
   - Land on homepage
   - Show magnetic text hover
   - Let "To Be Revealed" scramble and reveal
   - Pause on the gradient background

2. **Featured Work (7-15s)**
   - Smooth scroll down
   - Hover over project card (show 3D tilt)
   - Click to open case study

3. **Case Study (15-35s)**
   - Show smooth page transition
   - Scroll through content
   - Highlight Lighthouse scores
   - Show architecture diagrams

4. **Navigation (35-45s)**
   - Click back or use sidebar nav
   - Quick peek at About page
   - Show hover image effects

5. **Closing (45-60s)**
   - Scroll to footer
   - End on Contact section or "Available for opportunities"

## OpenScreen Features to Use

### Manual Zooms
- Zoom into "98/100 Performance" Lighthouse score

- Zoom into the animated hero text

### Annotations
- Add arrow pointing to "To Be Revealed" 👀
- Add text overlay: "Custom Animations"
- Highlight key metrics

### Cropping
- Crop out browser address bar if needed
- Focus on just the portfolio content

### Trimming
- Cut out any mistakes or long pauses
- Keep it tight and engaging

## Export Settings

### For LinkedIn Feed Post
- **Format:** MP4
- **Resolution:** 1920x1080
- **Aspect Ratio:** 16:9
- **Duration:** 30-60 seconds (max 90s)
- **File Size:** Under 200MB for fast upload

### For LinkedIn Profile Video (optional)
- **Format:** MP4
- **Resolution:** 1080x1080 (1:1 square)
- **Duration:** 3-10 seconds
- **Focus:** Just the hero section animation

## Pro Tips

### Visual Quality
- Record on a clean desktop (hide icons)
- Use dark mode if your portfolio supports it
- Ensure good lighting on screen recording
- Use motion blur for smooth panning

### Engagement
- Start with a hook (the animated hero grabs attention)
- Show, don't tell (let the animations speak)
- End with a clear CTA (link in bio/comments)
- Post during peak hours (Tue-Thu, 10am-12pm)

### Audio (optional)
- LinkedIn autoplay is muted by default
- Consider adding subtle background music
- Or add captions/text overlays instead

## Troubleshooting

### OpenScreen Issues
- **App won't open:** Run the quarantine command again
- **Screen recording blocked:** Check System Preferences → Security
- **Laggy recording:** Close other apps, record at 30fps instead
- **Large file size:** Export at 720p or use compression

### Browser Issues
- **Animations stuttering:** Close background tabs
- **Custom cursor not showing:** It's CSS-based, should appear
- **Fonts look weird:** Clear cache and hard reload (Cmd+Shift+R)
