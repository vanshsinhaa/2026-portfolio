# Technology Logos

Add your technology logo SVGs here.

## Required Logos:

Place these SVG files in this folder:

- `python.svg` - Python logo
- `javascript.svg` - JavaScript logo
- `react.svg` - React logo
- `nextjs.svg` - Next.js logo
- `typescript.svg` - TypeScript logo
- `nodejs.svg` - Node.js logo
- `docker.svg` - Docker logo
- `kubernetes.svg` - Kubernetes logo
- `gcp.svg` - Google Cloud Platform logo
- `bigquery.svg` - BigQuery logo
- `postgresql.svg` - PostgreSQL logo
- `tailwind.svg` - Tailwind CSS logo

## Where to find logos:

1. **Simple Icons** - https://simpleicons.org/ (SVG downloads)
2. **Official brand sites** - Most companies provide logo downloads
3. **SVG Repo** - https://www.svgrepo.com/
4. **Devicon** - https://devicon.dev/

## Tips:

- Use monochrome or simplified versions when possible
- Recommended size: 64x64px or scalable SVGs
- Keep file sizes small (<10KB each)
- Use consistent styling (all outline or all filled)

## To add/remove technologies:

Edit `/components/tech-carousel.tsx` and modify the `technologies` array.
